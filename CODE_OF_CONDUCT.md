# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境,作为贡献者和维护者,我们承诺让参与本项目和社区的每个人都不受骚扰,无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

## 我们的准则

营造积极环境的行为示例:
- 对他人友善友善
- 尊重不同观点和经验
- 给予建设性反馈
- 接受责任,向受我们错误影响的人道歉

不可接受的行为:
- 使用性化语言或图像
- 钓鱼、侮辱性评论、人身或政治攻击
- 公开或私下骚扰
- 未经明确许可泄露他人私人信息

## 执行

辱骂、骚扰或其他不可接受行为可通过 [GitHub Issues](https://github.com/howiitlee/fileguard/issues) 报告。所有投诉都会被及时审查和调查。

## 出处

本行为准则改编自 [Contributor Covenant][homepage] 2.1 版。
